# Hand Gesture Recognition (Python)

This is a beginner-friendly computer vision project built using Python, OpenCV, and MediaPipe.

The project detects hand landmarks using a webcam and counts the number of fingers shown using simple rule-based logic (no machine learning training involved).

This project was built as part of learning:
- Python basics
- Computer vision
- Rule-based AI logic
- Hand gesture recognition

---

## Features
- Real-time webcam input
- Hand landmark detection (MediaPipe)
- Finger counting (including thumb)
- Clean and simple Python logic

---

## Technologies Used
- Python
- OpenCV
- MediaPipe
- NumPy

---

## How It Works (Simple Explanation)
1. Webcam captures video frames
2. MediaPipe detects 21 hand landmarks
3. Finger positions are compared using coordinates
4. Rules decide whether a finger is open or closed
5. Total open fingers are displayed on screen
